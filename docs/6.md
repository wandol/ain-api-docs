# **6. 정보검색**
##  **개요**

* 검색엔진에 사용자가 입력한 검색어에 대한 검색 결과를 가져온다.

##  **전제조건**

* 색인데이터 필요

##  **정보**

- 요청 METHOD : GET
- 요청 URL : http://IP:PORT/konantech/search.do
- 호출 예시 : http://[IP:PORT]/konantech/search.do?kwd=야구&volume=volume1
- 파라미터

    |파라미터    |설명  |필수/선택  |
    |---------|---------|---------|
    |kwd     | 입력 데이터|필수|
    |volumeNm |  사용자 사전이름(사용자콘솔 확인)|필수|
