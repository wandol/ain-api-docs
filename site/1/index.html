    1. 형태소 분석¶

    개요¶

    • 입력 된 텍스트에 대한 형태소 분석 결과를 출력한다.(코난 자체 사전)

    전제조건¶

    • 색인 데이터 불 필요

    정보¶

    • 요청 Method : GET
    • 요청 URL : http://[IP:PORT]/konantech/kma.do
    • 호출 예시 : http://[IP:PORT]/konantech/kma.do?text=동해물과백두산이

    • 파라미터

      파라미터 설명 필수/선택
      text 입력 데이터 필수
